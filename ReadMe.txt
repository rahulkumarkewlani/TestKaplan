

Download zip project "Kaplan Test"
extract it.

Import that project in eclipse

*****Run project within eclipse
Go to package com.qa.testcases and run GmailLoginAndValidationTest.java as TestNG test.

Below actions script will perform.

1. Login into Gmail app
2. It will Search for email "QATestMail" and click on the first mail.
3. After that first email will open and it will validate the subject "QATestMail" and validate email body "This is QA test mail"



**** To run project from command line

1. Go to cmd
2. Open project folder location on cmd
3. run command "mvn test"

I have also attached command line "build success" result with this repo. Kindly have a look.
