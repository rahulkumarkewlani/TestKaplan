

Download project "Kaplan Test"
extract it.

Import that project in eclipse

Go to package com.qa.testcases and run GmailLoginAndValidationTest

Below actions script will perform.

1. Login into Gmail app
2. It will Search for email "QATestMail" and click on the first mail.
3. After that first email will open and it will validate the subject "QATestMail" and email body "This is QA test mail"
